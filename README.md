# VHDL-8-function-4-bit-ALU
This project implements an 8 function, 4 bit ALU using structural VHDL code. The functions are shown in the table below

| S1 | S0 | Cin | Function |
|----|----|-----|----------|
| 0 | 0 | 0 | Transfer A  |
| 0 | 0 | 1 | Increment A |
| 0 | 1 | 0 | A + B       |
| 0 | 1 | 1 | A + B + 1  |
|
